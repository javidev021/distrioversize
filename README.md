
# Propuesta: Integración de Desarrollo de Software en DistriOversize

## 1. Plataforma E-commerce Personalizada (Recomendado)
**Descripción:**  
Desarrollar una tienda online desde cero (no usar plantillas preconstruidas como Shopify o WooCommerce) utilizando tecnologías como **HTML, CSS y JavaScript** para el frontend, y **JavaScript (Node.js)** para el backend, con una base de datos **MySQL**.

**Cómo integrarlo al proyecto:**  
- Reemplazar la estrategia de "venta por redes sociales" por una plataforma propia de e-commerce.  
- Incluir funcionalidades avanzadas como:  
  - Personalización de diseños en tiempo real.  
  - Sistema de recomendaciones basado en preferencias.  
  - Integración con APIs de pasarelas de pago (MercadoPago, Wompi).  
  - Panel administrativo para gestionar inventario, pedidos y clientes.  

**Beneficios:**  
- Demuestra habilidades técnicas en desarrollo web full-stack.  
- Agrega valor al negocio al ofrecer una experiencia única al cliente.  
- Diferenciador clave frente a competidores que usan soluciones genéricas.  

---

## 2. Aplicación Móvil (Versión Web Responsive)
**Descripción:**  
En lugar de desarrollar apps nativas, se implementará un diseño **responsive** para que la tienda online pueda ser usada desde dispositivos móviles con una experiencia optimizada.  

**Ventajas:**  
- Menores costos de desarrollo.  
- Compatible con cualquier dispositivo con navegador.  
- Permite incluir funciones como notificaciones a través de la web.  

---

## 3. Sistema de Gestión Interna (Backoffice)
**Descripción:**  
Crear un sistema web para gestionar:  
- Inventario en tiempo real.  
- Pedidos y envíos.  
- Comunicación con clientes (integración con WhatsApp Business API o correos automatizados).  
- Reportes básicos de ventas.  

**Tecnologías:**  
- Frontend: HTML, CSS, JavaScript.  
- Backend: Node.js (JavaScript).  
- Base de datos: MySQL.  

**Cómo integrarlo:**  
- Incluirlo en la sección de "Operaciones" o "Logística" del plan de negocio.  
- Destacar cómo esta herramienta optimizará los procesos y reducirá costos.  

---

## 4. Integración con Redes Sociales mediante APIs
**Descripción:**  
Desarrollar un sistema que:  
- Automatice la publicación de nuevos productos en Instagram y Facebook.  
- Sincronice inventario entre la plataforma de e-commerce y redes sociales.  

**Tecnologías:**  
- APIs de Instagram Graph API y Facebook Marketing API.  
- JavaScript para las integraciones.

## Marco de Trabajo Scrum

### Roles
- **Product Owner (PO):** Responsable de definir los requerimientos y priorizar el Product Backlog.  
- **Scrum Master (SM):** Facilita el proceso Scrum, elimina impedimentos y asegura el cumplimiento de la metodología.  
- **Equipo de Desarrollo:** Grupo multidisciplinario encargado de construir el producto (frontend, backend, base de datos).  

### Artefactos
- **Product Backlog:** Lista priorizada de funcionalidades del e-commerce, backoffice e integraciones.  
- **Sprint Backlog:** Conjunto de tareas seleccionadas para un Sprint específico.  
- **Incremento:** Versión funcional del producto entregada al final de cada Sprint.  

### Ceremonias
- **Sprint Planning:** Reunión para planear qué se desarrollará en el Sprint.  
- **Daily Scrum:** Reuniones cortas diarias para sincronizar el progreso.  
- **Sprint Review:** Revisión del incremento al final de cada Sprint.  
- **Sprint Retrospective:** Reflexión del equipo para mejorar en el siguiente Sprint.  

### Cronograma de Sprints
- **Sprint 0 (2 semanas):** Configuración de entorno, repositorio GitHub, estructura del proyecto.  
- **Sprint 1 (3 semanas):** Desarrollo de frontend inicial (HTML, CSS, JavaScript) + diseño responsive.  
- **Sprint 2 (3 semanas):** Implementación backend con Node.js y conexión a MySQL.  
- **Sprint 3 (3 semanas):** Funcionalidades de e-commerce (carrito, pagos, personalización).  
- **Sprint 4 (2 semanas):** Desarrollo de Backoffice básico (gestión de inventario y pedidos).  
- **Sprint 5 (2 semanas):** Integración con APIs de redes sociales + pruebas finales.  
- **Sprint 6 (1 semana):** Despliegue y retrospectiva final.
